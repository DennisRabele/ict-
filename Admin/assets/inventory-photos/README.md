Place admin inventory images in this folder.

The dashboard currently references these files from `Admin/admin.js`:

- `laptop.svg`
- `server.svg`
- `switch.svg`
- `workstation.svg`
- `desktop.svg`
- `network.svg`

You can replace those files with your own JPG, PNG, WEBP, or SVG assets and update the photo list in `Admin/admin.js` if you add new filenames.
